# MIA Stil – Kalkulator Korpusa (Unified V5)

Spojeno: Wizard koraci + Loader u sidebaru.
1) Definiraš dimenzije/sklapanje
2) Odabereš materijale i usluge
3) Fronta
4) Rad i marža
5) Sažetak + izvoz (CSV, PDF)

## Pokretanje
pip install xlsxwriter
pip install -r requirements.txt
streamlit run app_unified_v5.py

## Fontovi za PDF
Za dijakritiku dodaj u `fonts/`:
- DejaVuSans.ttf
- DejaVuSans-Bold.ttf
